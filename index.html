01 — Technical Audit

Website Technical Analysis

A deep inspection of vivace.co.ke reveals 4 critical areas. Fixing these in Month 1 is the single highest-leverage action for ranking gains.

⚡ Page Speed & Core Web Vitals

⚠ Fix First

Images are lazy-loaded via JavaScript, causing Largest Contentful Paint (LCP) to exceed 3 seconds on mobile — a direct ranking penalty. No defer/async on render-blocking scripts detected. Google's Page Experience update makes this a hard ranking factor.

Convert all hero & service images to WebP format (30–50% size reduction)
Implement critical CSS inlining; defer non-critical stylesheets
Enable browser caching via .htaccess or WP caching plugin (WP Rocket)
Compress images to <150KB before upload; use lazy loading attribute
Target LCP < 2.5s, CLS < 0.1, FID < 100ms

📱 Mobile & UX

✓ Responsive

The site uses a responsive WordPress theme and the floating WhatsApp + Book Now CTAs are mobile-appropriate. However, tap targets should be verified ≥48px across all devices, and no horizontal scroll should occur on sub-pages.

Audit all tap targets using Chrome DevTools mobile emulator
Ensure font sizes ≥16px on mobile to prevent zoom triggers
Test WhatsApp CTA on all breakpoints (320px – 428px)
Check sub-pages (V-Lase, V-Shape) for overflow on narrow screens

📝 On-Page SEO & Meta

⚠ Critical Gap

Branded service page names (V-Lase, V-Shape, V-Glow, V-Derma) are the single biggest SEO mistake on the site. Google cannot map these to user queries like "laser hair removal Nairobi." Meta descriptions are absent on several pages. No structured data markup detected site-wide.

Rename all page <title> tags: "V-Lase" → "Laser Hair Removal Nairobi | Vivace Clinic"
Rewrite every H1 to include the primary target keyword + "Nairobi"
Write unique meta descriptions (150–160 chars) for every page
Install RankMath SEO plugin and run a site-wide audit
Add LocalBusiness, MedicalBusiness & Service schema to all pages

🔍 Crawlability & Indexation

⚠ Verify

As a WordPress site, Vivace likely has an auto-generated sitemap but it needs manual verification and submission to Google Search Console. The robots.txt must be checked for unintentional page blocks. Canonical tags are critical to avoid duplicate content from tag/category pages.

Set up and verify Google Search Console — submit XML sitemap immediately
Inspect robots.txt: ensure all service pages are crawlable
Set canonical tags on all treatment pages to prevent duplication
Fix any 404 errors or broken internal links found in GSC
Ensure HTTPS is forced on all pages (no mixed content warnings)

04 — Beyond Keywords

The Full SEO Toolstack

Keywords alone don't rank websites. Top-3 positions require mastery across 6 distinct pillars — each with its own tools, tactics, and measurement approach.

Most clinics optimise for keywords and stop. The ones who reach Top 3 and stay there win because they operate a complete system: technical hygiene, authority signals, local dominance, content depth, conversion alignment, and data-driven iteration. This is that system for Vivace.

Pillar 1

Technical SEO Tools

The foundation. If Google can't crawl, index, and render your site correctly, no keyword strategy matters. These tools identify and fix structural issues.

Google Search Console — free crawl data, impressions, clicks, index coverage
Screaming Frog SEO Spider — full site crawl: broken links, redirects, duplicate meta

GTmetrix — waterfall load analysis, speed optimisation recommendations

05 — Gap Analysis

Content & Ranking Gaps

What competitors have that Vivace doesn't — and the exact opportunities to close those gaps faster than they can react.

❌ Critical Gaps

Branded Page Names Blocking Rankings

V-Lase, V-Shape, V-Glow, V-Derma are invisible to Google's keyword matching. These must be renamed in titles, H1s, and meta — but URLs can stay to preserve any existing link equity.

No Weight Loss or Weight Management Page

"Weight loss clinic Nairobi" gets 500–900 monthly searches. There is no dedicated landing page for this keyword cluster. Competitors with even basic pages will rank above Vivace.

No Standalone Microneedling or Skin Boosters Page

Both are growing search terms in Nairobi's market. Without dedicated pages, the treatment can't rank — it's buried under generic category pages.

Zero Blog Content

Informational queries ("how many laser sessions for dark skin," "botox aftercare Nairobi") funnel high-intent visitors. No blog means Vivace captures zero of this traffic. Competitors with even 5 blog posts have a significant advantage.

No FAQ or Schema Markup

FAQ schema can trigger rich results in Google — expanded listings that take up 2–3× the screen real estate. Currently zero schema is detected across the entire site.

✓ Quick Wins & Opportunities

Rename Page Titles & H1s (Week 1 — highest ROI)

Zero content work required. Simply updating title tags and H1s with target keywords can move existing pages from page 4 to page 1 within 4–8 weeks. This is the fastest ranking lever on the entire site.

Create 3 Missing Service Pages (Month 1)

Weight Loss, Microneedling, and Skin Boosters pages. Each needs 600+ words of optimised content, an FAQ section, a booking CTA, and LocalBusiness schema. These can rank within 60 days.

GBP Optimisation — Map Pack Entry (Weeks 1–3)

Vivace's 4.9★ rating gives it a strong head start. Adding all 11 services with keyword-rich descriptions, weekly GBP posts, and 20+ new treatment photos can push it into the local Top 3 faster than organic results.

FAQ Schema on All Service Pages (Month 1)

Using RankMath's FAQ block, adding 4–6 questions per service page takes 30 minutes per page and can generate rich results within 2–4 weeks of implementation.

Review Generation Campaign (Ongoing)

Send post-appointment WhatsApp message with a direct GBP review link. Reaching 200+ reviews with a maintained 4.9★ rating is a strong local ranking signal no competitor can fake or catch quickly.
